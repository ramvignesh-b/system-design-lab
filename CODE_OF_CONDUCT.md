# Code of Conduct

## Our Pledge
We want this repository to be a welcoming learning space for everyone,
regardless of experience level, identity, or background.

## Expected Behavior
- Be respectful and constructive in issues and pull requests.
- Assume good intent and ask clarifying questions before criticizing.
- Share evidence when suggesting technical changes.
- Keep feedback specific, actionable, and kind.

## Unacceptable Behavior
- Harassment, hate speech, or personal attacks.
- Doxxing or sharing private information.
- Repeated disruptive behavior after feedback.

## Scope
This Code of Conduct applies to all interactions in this repository, including
issues, pull requests, and discussions.

## Reporting
If you experience or witness unacceptable behavior, open a private report with
the maintainers via repository contact details.

## Enforcement
Maintainers may remove comments, lock threads, or block participation when
behavior violates this Code of Conduct.
