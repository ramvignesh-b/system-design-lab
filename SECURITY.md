# Security Policy

## Supported Scope
This repository is a learning and demo monorepo. Labs are intentionally
minimal and not production hardened.

## Reporting a Vulnerability
Please report potential vulnerabilities privately to maintainers first. Include:
- A clear description of the issue
- Affected files/labs
- Steps to reproduce
- Potential impact

Avoid opening public issues for unpatched security problems.

## Response Expectations
Maintainers will acknowledge reports and provide a remediation plan when the
issue is reproducible and relevant to this repository.

## Notes
Credentials and settings in labs are demo-only and must not be reused in
production systems.
